### Java Testing Developer
1.	¿Qué es Java y cómo funciona Java?
2.	Programación orientada a objetos
3.	Sintaxis esencial de Java
  a.	Condicionales (if bloque/oneline & switch clásico/moderno)
  b.	Iteradores (for, while, do/while, recursividad)
  c.	Declaración de variables, atributos, métodos y constructores
  d.	Inyección de dependencias
  e.	Anotaciones
  f.	Manejo de errores
  g.	Lambdas & Streams
4.	Motor de testing
  a.	Unidad básica (Test)
  b.	Ciclo de vida del test (BeforeAll, BeforeEach, AfterEach, AfterAll)
  c.	Validaciones de exito (metodos de clase Assertions)
  d.	Nombres descriptivos 
5.	Mockito
  a.	Diferencia teórica entre dummy/stub y mock
  b.	Inyección de dependencias (Mock & InjectMocks)
  c.	Inyección de comportamiento
  d.	Triggers de verificación
6.	¿Cómo funciona Spring y Spring Boot? ¿Cómo probarlo?
  a.	Simulación de llamado al API desde afuera
  b.	Simulación de llamado a base de datos con H2
  c.	Inyección de Mocks
7.	Métricas de aseguramiento de calidad
  a.	Coverage
  b.	Edge cases
